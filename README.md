# A Villager's Dream
[![Build Status](https://github.com/Priestytheplushie/AVillagersDream/actions/workflows/build.yml/badge.svg)](https://github.com/Priestytheplushie/AVillagersDream/actions)
![Minecraft Version](https://img.shields.io/badge/Minecraft-1.21.1-brightgreen?style=flat-square)
![Loader](https://img.shields.io/badge/Loader-NeoForge-orange?style=flat-square)

A Villager's Dream is a Minecraft Neoforge 1.21.1 mod which adds brand new Emerald and Villager themed content. 

> [!WARNING] 
> **Early Access**: This mod is still in development, If you find any bugs, report them in Issues

## ✨ Features
* **Emerald Tools:** Brand new set of **Emerald Tools** with unique Abilities.
* **Rubies:** The original, scrapped Villager currency, back with new, fiery powers.
* **Raiders:** Brand new **Scouts** and **Disguised Illagers** to improve Village and Raid encounters.
* **Compatibility:** Works with NeoForge 1.21.1.

## 🛠️ Development
This mod is developed using **MCreator**. 

> [!TIP]
> You don't need to build the mod yourself! You can always find the latest "Nightly" version in the **Releases** tab or the latest stable "Ready-to-Play" version on modrinth

* **To Edit:** Download the repository and open the `.mcreator` file in the MCreator IDE.
* **To Build:** Run `./gradlew build` or check the GitHub Actions tab for automated builds.
